Vagrant Cluster
=============
### Setup
#### Install Vagrant
* http://www.vagrantup.com/downloads.html
* http://docs.vagrantup.com/v2/

#### Install Vagrant Plugins
```bash
vagrant plugin install vagrant-hostmanager
vagrant plugin install vagrant-cachier
```

#### Vagrant Up
```bash
vagrant up
```
#### Root Password
* rootpw
